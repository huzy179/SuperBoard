# SuperBoard
SuperBoard: Nền tảng quản lý dự án real-time chuẩn Enterprise, kết hợp sức mạnh của Jira, Notion và Slack. Dự án fullstack ứng dụng kiến trúc Monorepo (Next.js, NestJS, Python), xử lý đồng thời (Redis/WebSockets), AI thông minh, phân tích dữ liệu (ClickHouse) và hạ tầng Cloud Native (Docker/K8s) cho hiệu năng tối đa.
